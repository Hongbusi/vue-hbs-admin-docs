## 开发要求

- Nodejs version 14+
- [pnpm](https://github.com/pnpm/pnpm)

## 领取任务

- 项目成员：对应 issue 添加 `label`，格式为：`Developer:username`。
- 非项目成员：对应任务 issue 下留言，会为你添加 `label` -> `Developer`。

## 判断任务是否已被领取

- 当 issue 存在 `Developer` 或格式为 `Developer:username` 的 `label`，既已被领取。
- 否则，视为未被领取。
